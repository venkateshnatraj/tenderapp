# tenderapp

## steps to start the app
   - #####  `npm i`
   - #####  `npm run dev` for development
   - ##### `npm start` for production

## Docker
   - #### docker build -t tender-app-v4 .
   - #### docker run --name tender-app-ct-v1 -p 3000:80 -d tender-app-v4
   - #### virtual box  : http://192.168.99.100:3000/
